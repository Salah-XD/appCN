# @app-cn/web

The appCN docs site — landing, component catalog, `/docs/cli`, and the
installable shadcn registry served at `/r/<slug>.json`.

Deployed to <https://appcn.vercel.app>.

## Dev

```bash
pnpm --filter @app-cn/web dev          # http://localhost:3100
pnpm --filter @app-cn/web typecheck
pnpm registry:build                    # rebuild apps/web/public/r/*.json
```

`/r/*.json` is gitignored — regenerated by `pnpm registry:build`, which also
runs the validator (`scripts/validate-registry.mjs`).

## How it fits

- **Source of truth** for components: `packages/ui/src/`. Nothing here lives there.
- **Live previews** iframe the Expo showcase (`apps/showcase`), so they
  exercise the exact same code that runs on a phone.
- **No React Native deps.** This app is plain Next.js + Tailwind v3.
